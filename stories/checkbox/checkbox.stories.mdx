## Checkbox Group Inputs

| 名称      | 类型              | 默认值 | 描述               |
| --------- | ----------------- | ------ | ------------------ |
| direction | 'row' \| 'column' | 'row'  | 按钮横纵向排列方式 |
| trackFn   | Function          | -      | 值匹配规则         |

## Checkbox Inputs

| 名称     | 类型    | 默认值 | 描述       |
| -------- | ------- | ------ | ---------- |
| disabled | boolean | false  | 是否禁用   |
| label    | string  | -      | 选项对应值 |

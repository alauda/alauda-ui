# Update Icons

AUI 图标托管在 [IconFont 项目](https://www.iconfont.cn/manage/index?spm=a313x.7781069.1998910419.11&manage_type=myprojects&projectId=1625966&keyword=&project_type=&page=)，超级管理员为于雷，如需加入项目请将账号昵称发给他。

## 更新步骤

1. 进入[项目](https://www.iconfont.cn/manage/index?spm=a313x.7781069.1998910419.11&manage_type=myprojects&projectId=1625966&keyword=&project_type=&page=)点击更新代码
2. 点击 `下载至本地`，下载图标并解压
3. 复制 `iconfont.js` 文件内 svg 文本并替换 AUI 仓库 `src/icon/icons.ts` 文件中 svg 文本
4. 提交代码，发布新版本
